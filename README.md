## 前言

上一个网络安全产品[《魔盒安全情报》][1]会不定时给我推送各种最新的漏洞情报，于是就写了一个全自动化漏洞扫描系统。

主要原理是通过空间测绘平台和搜索引擎自动采集目标资产，然后再通过封装好的漏洞插件扫描漏洞，最后将结果保存到数据库，再通过系统展示出来。
